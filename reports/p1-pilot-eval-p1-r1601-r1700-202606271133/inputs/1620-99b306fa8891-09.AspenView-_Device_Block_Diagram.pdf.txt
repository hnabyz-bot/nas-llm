2.2 Dependency Description
1) Worklist Module
A. Worklist Module
And then change over from Worklist Mode to Exam Mode for X-ray exposure.
B. Database Module
Query / Save / Edit / Delete patient information in Local database.
C. MWL(Modality Worklist) module
MWL module searches for patient information from OCS server or PACS Worklist Server.
2) Exam (Decision Board) Module
A. Device Control module
Detector Control module acquire raw image form detector (Library).
Generator control module adjusts the X-ray dose and shoots (Library).
B. Pre-Image processing and display module
Image display module display raw image files on the monitor with correction.
C. Post-Image process module
Image processing module enhance image quality to diagnose.
D. DICOM Module
DICOM module make DICOM image file form raw image file and save.
E. Database Module
Save Study/Patient’s information
3) Viewer (Decision Board) Module
A. Image display module
Image display module display DICOM image files on the monitor.
B. Post process module
Image processing module enhance image quality to diagnose.
C. DICOM Module
DICOM module make DICOM image file form raw image file and save.
And then it send DICOM images to PACS server or print.
Also Export DICOM Image to external storage.
4) Browser (Worklist) Module
A. Patient management module
Patient management module can edit or delete data.
And it search patient easily to diagnose.
B. Database Module
Save / Edit study and patient information
Search Study information
Delete Study information
5) Database Module
Query / Save / Edit / Delete patient information in worklist mode
Query / Edit /Delete Study information in browser(worklist) mode
Save / Edit Image information in Exam(Decision Board) mode , Viewer(Decision Board) mode
