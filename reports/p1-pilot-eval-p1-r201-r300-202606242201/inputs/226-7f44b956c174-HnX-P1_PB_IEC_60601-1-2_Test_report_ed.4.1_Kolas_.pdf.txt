1.8 Basic Safety, Essential Performance and Immunity Pass/Fail Criteria
as determined by the Manufacturer
Description of Basic Safety and Essential Performance
The HnX-P1 portable X-ray System is a medical device intended for use by a qualified/trained physician or
technician for the purpose of acquiring X-ray images of the desired parts of patient’s anatomy (including
head, cervical spine, chest, abdomen, lumbar spine, pelvis and extremities).
This system is not intended for mammography.
1. X-ray output must be within ±10% of the set value.
(Tube voltage, Tube current, Irradiation time)
2. The output of the X-ray must be within ±10% of the set value even when the input power
fluctuates.
(Tube voltage, Tube current, Irradiation time)
3. Bluetooth functionality must operate without issues and function properly.
Description how the Basic Safety and Essential Performance were monitored during each test
1. Measure the output during X-ray exposure.
(Tube voltage, Tube current, Irradiation time)
2. Measure the X-ray output during power fluctuations.
3. Verify that the Bluetooth functionality operates properly.
Immunity Pass/Fail Criteria
1. Whether the X-ray exposure exceeds the set value by ±10%.
2. Whether the X-ray exposure exceeds the set value by ±10% during input power fluctuations.
3. Whether the Bluetooth function operates properly.

The major components of the X-ray main unit
include handle, enclosure, main control panel,
system control board, high-voltage tank,
inverter, collimator (beam limiter), and system
control software running on the system control
board. The system control software is for real-
time interaction and control with various circuit
modules inside the X-ray generator. The
software responds to user operations on the
control panel. The user can adjust and control
the kV and mAs parameters, and the software
will display the parameters or directly load the
APR parameters. The software loads the
control data from X-ray output into the high-
voltage generation control circuit of the system
control board, and control the high-voltage
tank to generate high-voltage to excite the X-
ray tube inside to emit X-rays, control the
switch of the collimator, and monitor the
working status of the device, and control the
display of the status indicators.
The measured X-ray output value is within
±10% of the set value (tube voltage, tube
current, and exposure time).
