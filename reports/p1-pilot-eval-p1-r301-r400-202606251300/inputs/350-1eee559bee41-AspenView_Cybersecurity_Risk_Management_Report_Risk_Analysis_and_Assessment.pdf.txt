5. Cyber Security Risk Analysis
5.1 Severity evaluation criteria
Score Severity Possible description
1 Catastrophic Results in patient death
2 Critical Results in permanent impairment or life-threatening injury
3 Serious Result in injury or impairment requiring professional medical intervention
4 Marginal Results in temporary injury or impairment not requiring professional medical intervention
5 Negligible Inconvenience or temporary discomfort
5.2 Definition of Security Test Methods
Security Test
Method Definition
Vulnerability
Scanning
Type of security testing that helps perform analysis of vulnerabilities and misconfiguration
across the operating system, web server, network, system, and application.
White Box Test White box testing focuses on understanding how the application works, then attempt to break
into the application through knowledge of the source code or the software behavior
Network Security
Test
Type of penetration testing that focuses on evaluating a computer network's vulnerability to
intrusion
Penetration Test Penetration testing is a form of security testing wherein security engineers simulate a hack to
check vulnerabilities present in a site, an application, or a network.
