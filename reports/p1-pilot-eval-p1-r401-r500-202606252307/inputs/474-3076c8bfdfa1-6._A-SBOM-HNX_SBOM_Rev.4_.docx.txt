Overview

Document Overview

This document aims to provide cybersecurity lifecycle risk analysis and threat modeling for Portable X-ray System(HnX-P1, HnX-PB).

Medical Device Overview

Product name: Portable X-ray System

Model name: HnX-P1, HnX-PB

Manufacturer: H&abyz Co., Ltd.

This product is a battery-powered portable X-ray system designed and manufactured by H&abyz. This portable X-ray system (model: HnX-P1, HnX-PB) consists of an LCD display with up and down soft keys for kV control, an X-ray generator (power transformer), an X-ray tube assembly, and a collimator.

Compared to conventional X-ray products, this device boasts a sophisticated structure, compact design, light weight, and easy operation.

The main components of the X-ray unit are the handle, enclosure, main control panel, system control board, high-voltage tank, inverter, collimator (beam limiter), and system control software running on the system control board. The system control software interacts with and controls the various circuit modules within the X-ray generator in real time. This software responds to user input from the control panel. Users can adjust and control kV and mAs parameters, and the software can display these parameters or directly load APR parameters. This software loads control data for the X-ray output into the high-voltage generation control circuit on the system control board, controls the high-voltage tank to generate high voltage, excite the internal X-ray tube, and emit X-rays. It also controls the collimator switch, monitors the device's operating status, and controls the display of status indicators.

It also allows users to view thumbnails of acquired images on the LCD via Bluetooth.

This product is intended for X-ray imaging and diagnosis in facilities with mobile or fixed locations. It can be mounted on a cart or stand and used with a film cassette or flat panel detector, a computer for signal reception and detection, and image processing software. (The cart, stand, film cassette, flat panel detector, and image processing software are not included in the basic product components.)

To ensure the mechanical stability and efficiency of the device, the recommended maximum load that the stand can safely withstand is 6 kg.

The cybersecurity risk of the HnX-P1 & HnX-PB have been addressed to assure that no new or increased cybersecurity risks were introduced as a part of device risk analysis. These risks are defined as sequence of events leading to a hazardous situation, and the control for their risks were treated and implemented as proposed in the risk analysis (e.g., requirements, verification).

The variant model HnX-PB has only changed the exterior design compared to the main model HnX-P1, and the interface, exterior material, internal structure, PCB, software, etc. are all the same.Therefore, no additional cybersecurity risk management is performed due to the addition of the variant model (HnX-PB).

2.2 EOL, EOS

SBOM security and vulnerability assessment method.

Security and vulnerability assessment method.

‘HnX_P1.hex’ and ‘HnX_PB.hex’ are the device firmware; threat modeling was conducted for each function, and both penetration testing and static code analysis were performed.

For all off‑the‑shelf and open‑source software and firmware components listed in this SBOM (e.g. STMicroelectronics, KISA), we performed threat modeling and Cybersecurity Risk Management. In addition, all COTS and open‑source components were cross‑checked against NIST’s NVD and the CISA KEV Catalog; no publicly known vulnerabilities affecting this product were identified to date.
