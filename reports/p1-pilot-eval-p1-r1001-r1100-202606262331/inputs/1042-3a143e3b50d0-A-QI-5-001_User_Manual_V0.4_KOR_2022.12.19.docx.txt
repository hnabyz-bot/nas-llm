 1.4 경고

심볼

설명

사용 및 저장 환경

사용자와 환자의 안전을 위해 본 설명서에 명시된 작동 지침을 따릅니다.

연결

디텍터를 지정된 것 이외의 것과 연결하지 않습니다. 그렇지 않으면 화재 또는 감전의 원인이 될 수 있습니다.

감전의 위험을 방지하려면 이 디텍터를 보호 접지가 있는 전원 공급 장치에만 연결해야 합니다.

신호 입력 부품/시그널 출력 부품과 환자를 동시에 만지지 않습니다. 전류 누출로 인한 감전 위험이 있습니다.

의료 전기 장비에 연결된 추가 장비는 해당 IEC 또는 ISO 표준을 준수해야 합니다. 또한 모든 구성은 의료 전기 시스템의 요구 사항을 준수해야 합니다. 추가적인 의료 전기 장비의 연결 시 연결하고자 하는 시스템이 의료 전기 시스템에 대한 요구 사항을 준수해야 합니다. 의료기기법이 위에 언급된 사항보다 우선합니다.

 디텍터와 환자 환경에 연결된 장비는 의학적으로 격리된 전원으로부터 전원을 공급받거나 의료적으로 격리된 장치여야 합니다. 격리되지 않은 소스로부터 전원을 공급받는 장비로 인해 누설 전류가 안전 수준을 초과할 수 있습니다. 절연되지 않은 콘센트에 연결된 액세서리 또는 장치에 의해 생성된 누설 전류가 디텍터의 누설 전류에 추가될 수 있습니다.

조작

디텍터를 사용하는 동안 항상 시스템과 환자의 상태를 점검하여 정상인지 확인하십시오. 문제가 발견되면 필요에 따라 디텍터 작동을 중지하는 등 적절한 조치를 취하십시오.

화재 또는 감전의 원인이 될 수 있으므로 디텍터를 분해하거나 개조하지 마십시오.

또한 디텍터에는 감전의 원인이 될 수 있는 부품이 내장되어 있으므로, 만지면 사망이나 중상을 입을 수 있습니다.

디텍터를 치거나 떨어뜨리지 않습니다. 디텍터가 강한 충격을 받으면 손상될 수 있으며, 수리하지 않고 디텍터를 사용할 경우 화재나 감전의 원인이 될 수 있습니다.

문제 발생 시

다음과 같은 경우 즉시 각 디텍터의 전원을 끄고 AC 콘센트에서 전원 공급 코드를 뽑고 H&abyz 고객서비스팀 또는 공인 대리점에 문의합니다.

연기, 이상한 냄새 또는 비정상적인 소리가 있을 경우.

액체가 디텍터에 쏟아지거나 구멍을 통해 금속 물체가 들어간 경우.

디텍터가 떨어져 손상되었을 경우.

에러 메세지

디텍터 사용시 발생하는 에러에 대한 메시지는 포함된 소프트웨어에서 직접 출력하지 않습니다. 상위 레벨 소프트웨어에서 사용할 수 있도록 텍스트 형식으로 제공합니다. (1.6절 참고)

에러 메시지 내용 및 조치사항은 해당 상위 레벨 소프트웨어 매뉴얼을 참고하십시요.

상위 레벨 소프트웨어 매뉴얼로 해결이 불가능 할 경우 H&abyz 고객서비스팀 또는 공인 대리점에 문의합니다.

유지 관리 및 검사

안전상의 이유로 다음 검사를 수행할 때에는 반드시 디텍터의 전원을 꺼야 합니다. 그렇지 않으면 감전될 수 있습니다.

디텍터를 청소할 때에는 각 디텍터의 전원을 끄고 AC 콘센트에서 전원 케이블을 분리해야 합니다.

벤젠과 같은 용제는 사용하지 마십시오. 그렇지 않으면 화재 또는 감전이 발생할 수 있습니다.

IPA(아이소프로필 알코올) 또는 다른 액체와 직접 접촉하지 않도록 방수 장갑을 착용합니다.

디텍터의 유지보수 작업은 공인 서비스 공급자가 수행해야 합니다. 문제가 지속적으로 해결되지 않는 경우 화재 또는 감전의 원인이 될 수 있습니다.

환자 접촉

디텍터를 지정된 장소에 보관하고 환자를 디텍터에 직접 접촉하지 않습니다.

디텍터 장치의 케이블을 강하게 당기거나 케이블이 손상되는 경우 화재 또는 감전의 원인이 될 수 있습니다.

케이블이 누군가 발에 걸릴 수 있는 장소.

조작

감염 방지를 위해 IPA(아이소프로필 알코올) 액체를 적신 부드러운 천으로 CFRP(카본 섬유 강화 플라스틱)를 닦아주십시오. IPA 또는 다른 액체와 직접 접촉하지 않도록 방수 장갑을 착용합니다.

안전상의 이유로 디텍터를 사용하지 않을 때는 각 장비의 전원을 반드시 꺼야 합니다.

케이블의 위치

모든 케이블이 밟히거나 걸려 넘어지거나 손상 또는 압박을 받지 않도록 해야 합니다.

유지 관리 및 검사

안전상의 이유로 디텍터를 사용하기 전에 반드시 검사해야 합니다. 또, 적어도 1년에 한 번은 정기 점검을 실시합니다.

디텍터에 결함이 있는 경우에는 디텍터를 임의로 분해하지 않습니다. 디텍터의 유지보수는 공인 서비스 공급자가 수행해야 합니다. H&abyz 고객 서비스 팀 또는 공인 대리점에 문의하시기 바랍니다.

사용자 설명서에서 교체 가능한 구성 요소를 확인해야 합니다.

1.6 안전에 관한 정보

일반 사항

준비

케이블을 정해진 커넥터에 연결해야 합니다. 그렇지 않으면 디텍터가 오작동하거나 손상될 수 있습니다.

취급

강한 충격을 가하거나 떨어뜨리거나 받을 경우 디텍터가 손상될 수 있으므로 주의하여 취급합니다.

디텍터가 구부러지지 않도록 평평한 곳에서 디텍터를 사용해야 합니다. 그렇지 않으면 디텍터가 손상될 수 있습니다.

디텍터를 매일 점검하고 제대로 작동하는지 확인해야 합니다. 차가운 공간에서 방을 갑자기 가열하면 디텍터에 응결이 형성됩니다. 이 경우 응결이 사라질 때까지 기다렸다가 노출을 수행합니다. 디텍터에 응결이 형성된 상태에서 디텍터를 사용할 경우 디텍터의 품질에 문제가 발생할 수 있습니다. 에어컨을 사용할 때는 실내와 디텍터의 온도 차이가 발생하지 않도록 온도를 점진적으로 올리거나 낮춰 응결이 생기지 않도록 해야 합니다. 권장되는 실내 온도를 따릅니다.

강한 자기장을 생성하는 장치 근처에서 디텍터를 사용하지 않습니다. 이 경우, 영상에 잡음 또는 왜곡이 발생할 수 있습니다.

커넥터와 환자가 접촉하지 않도록 합니다.

커넥터는 외부 장치에 연결되도록 되어 있으며 IEC 표준을 준수해야 합니다.

디텍터에 과도한 중량을 가하지 않습니다. 그렇지 않으면 디텍터가 손상될 수 있습니다

 2.1 제품 기능

HAD1717MG 및 HAD1717MC는 디지털 방사선 시스템에 대해 보다 빠르고 능률적인 접근을 위하여 설계된 유선 디지털 평면 패널 디텍터입니다. HAD1717MG와 HAD1717MC 디텍터는 TFT glass 와 Scintillators(GDX, CsI)의 조합을 활용하며, 이들 및 전자 장치는 하나의 패키지에 들어 있습니다. 본 디텍터 제품은 generator integration 없이 사용할 수 있는 자동 방사선 신호 감지 기술을 지원합니다.

HAD1717MG 및 HAD1717MC의 평면 패널 센서는 비정형 실리콘 기술에 기반한 박막 기술을 사용하여 제작됩니다. 전기적으로, 이 센서는 기존의 포토다이오드 배열과 매우 유사합니다. 어레이의 각 픽셀은 동일한 전자 회로의 광 감지 포토다이오드 및 전환 박막 트랜지스터(TFT)로 구성됩니다. 비정질 실리콘 포토다이오드는 시각적 빛에 민감하며, 반응 곡선은 인간의 시각과 거의 유사한 성능을 가집니다. 또한 조사 선량에 따른 우수한 선형성을 가집니다.

제품 설치

 1) 통합

Auto Trigger(오토 트리거) 모드

제너레이터와 디텍터의 물리적인 연결 없이 사용하는 방식으로, 제네레이터가 방출하는 방사선을 디텍터가 자동으로 검출합니다. 따라서 디텍터(전원 공급 장치)와 제너레이터 사이의 물리적인 연결은 필요하지 않습니다.

※ Generator, PC는 본 구성품에 포함되지 않습니다.

Line(Manual) Trigger(유선 트리거) 모드

제너레이터와 디텍터 간의 물리적인 연결(Trigger Cable)을 하는 방식으로, 제네레이터가 동작 준비가 완료 되면 디텍터로 신호를 보내고, 이 신호를 디텍터가 수신하여 동작 준비가 완료 되면 제너레이터로 다시 신호를 보내 제너레이터의 방사선을 방출시킵니다.

※ Generator, PC는 본 구성품에 포함되지 않습니다.

Line(Manual) Trigger Mode 에서의 Trigger 연결

Trigger 케이블 커넥터는 아래 그림과 같이 배치됩니다.

 2) PC 설정

네트워크 설정은 아래와 같습니다.

[네트워크 연결]에서 [로컬 영역 연결]을 마우스 오른쪽 키로 클릭한 후 “속성”을 클릭합니다.

 [인터넷 프로토콜 버전 4(TCP/IP)] 선택하고 “속성”을 클릭합니다.

[다음 IP 주소 사용]을 선택한 후 디텍터 IP 주소를 입력합니다.

H&Abyz 디텍터의 기본 IP 주소는 2.2.22.101 이며 사용 가능한 IP는 2.2.22.10 ~ 2.2.22.101입니다. PC IP 주소의 경우 2.2.22.102 ~ 2.2.22.254, subnet mask 의 경우 255.255.255.0을 권장합니다.

 3) 영상 획득 소프트웨어(XAS) 설정

디텍터를 연결하고 전원을 켭니다.

연결된 디텍터 IP 주소가 IP 주소와 다른 경우 “Detector IP Address” 에 연결된 디텍터 IP 주소를 입력하고 “set”을 클릭합니다.

“C:\XAS” 에 위치한 “XAS.exe”를 실행합니다. 디텍터가 PC에 연결되면 소프트웨어의 네트워크 상태가 빨간 X 자에서 녹색 체크로 변경됩니다.

연결을 확인한 후, Line Trigger 와 Auto Trigger 중 사용하고자 하는 Trigger 모드를 선택합니다.

Calibration” 탭을 선택하고 “Common Setting” 을 클릭합니다.

“Window Time”에 값을 설정해야 합니다. Window Time 은 100msec 이상이고 최소 단위는 100msec여야 합니다. ADC resolution은 14bit와 16bit 중 선택이 가능합니다. “En_16bit”를 선택하면 디텍터에서 16bit 영상 데이터가 획득됩니다. ADC resolution을 선택한 후 “Set Setting”을 클릭합니다.

“Window Time”은 X-ray 조사 시간보다 짧으면 영상이 제대로 획득되지 않습니다.

ADC resolution을 선택한 후 “Set Setting”을 클릭해야 합니다.

 3.2 보정

영상을 적절하게 획득하려면 보정을 수행해야 합니다. 보정하지 않으면 최적의 영상을 획득할 수 없습니다.

H&abyz는 전원을 켠 후 15분간 디텍터를 예열할 것을 권장합니다.

보정은 일반적으로 6개월에 한 번 또는 X-ray 소스의 중앙 빔이 디텍터에 상대적으로 이동될 때마다 정기적으로 수행해야 합니다.

Dark 영상 획득

“Calibration”탭에서 Dark 영상을 획득하려면 “Get Dark”를 클릭합니다. “Preview”창이 나타나고 Dark 영상이 획득됩니다. 획득된 Dark 영상이 영상 파일 목록에 표시됩니다.

H&abyz는 영상 성능을 향상하기 위해 “Average Dark” 옵션 사용을 권장합니다.

“Average Dark”옵션을 선택한 경우 Dark 영상이 “total dark frame”의 값 만큼 자동 획득됩니다.

Bright 영상 획득 및 보정 포인트 영상 생성

Bright 영상을 획득하기 위해 “Get Bright”를 클릭합니다. Bright 영상은 각 4개의 cal point에서 획득해야 합니다. 획득한 Bright 영상은 영상 파일 목록에 표시됩니다.

H&abyz는 다음 표로 4개의 cal point를 권장합니다. 보정 지점이 충분하지 않으면 일부 artifact 가 획득된 영상에 표시될 수 있습니다.

Auto Trigger 모드에서는 X-ray 조사 시간이 30msec 보다 길어야 합니다. 30msec 보다 짧은 X-ray 조사 시간에서는 영상을 획득할 수 없습니다.

BPM 탭에서 “Generate BPM” 을 클릭하고 팝업 창에서 “OK” 를 클릭하여 BPM 생성을 완료합니다.

4. 사용법

4.1 설정

영상 설정

영상 크기를 변경하려면 “Setting” 탭의 “Image Crop”을 사용합니다. 그리고 “Save”를 클릭합니다.

“Acquisition” 탭에서 “Image Crop” 옵션을 선택한 경우 획득한 영상을 잘라낼 수 있습니다.

선택적으로 Processing 옵션을 선택할 수 있습니다. 그러나 모든 Processing 옵션을 선택합니다.

 “Acquisition” 탭의 모든 Processing 옵션을 선택한 경우 보정된 영상을 획득할 수 있습니다.

다중 디텍터 설정

H&abyz 디텍터는 여러 디텍터를 하나의 PC에 연결하는 “다중 디텍터” 기능을 지원합니다. 이 기능은 두 개의 디텍터가 X-ray 시스템에 통합되어 있을 때 유용합니다.

PC 상태에 따라 다른 디텍터로 변경하는 데 몇 초 정도 걸릴 수 있습니다. 디텍터를 변경하는 동안 고객이 반복적으로 다른 디텍터를 선택하는 경우 모든 디텍터들은 분리될 수 있습니다..

 “Setting” 탭의 “Edit”를 클릭합니다.

 “Add”를 클릭합니다. 추가할 수 있는 디텍터가 “Detector List”에 나열됩니다.

추가된 디텍터 이름을 “Alias”에 입력하고 IP 주소와 영상 크기를 변경합니다. 구성 변경을 적용하려면 “Apply”를 클릭합니다. 구성이 완료되면 “Confirm”을 클릭합니다.

아래 그림과 같이 다른 디텍터가 추가되었는지 확인하고 필요한 디텍터를 선택합니다.

 4.2 영상 획득

네트워크 연결 상태와 필요한 Processing 옵션이 “Acquisition” 탭에 선택되어 있는지 확인합니다.

 “Image Name” 에 영상 파일 이름을 입력하고, “Get Image”을 클릭하여 영상을 획득합니다. 획득된 영상은 영상 파일 목록에 표시됩니다.

Auto Trigger 모드에서는 X-ray 조사 시간이 30msec 보다 길어야 합니다. 30msec 보다 짧은 X-ray 조사 시간에서는 영상을 획득할 수 없습니다.

“Preview” 팝업 창에서 “Busy” 메시지가 표시될 때에는 X-ray를 조사하지 마십시오. X-ray가 조사되더라도 “Busy” 상태에서는 영상을 획득할 수 없습니다. “Preview” 창에 “Ready for Exposure” 가 표시되면 X-ray를 조사합니다.

5. 관리

 5.1 청소

디텍터가 오염된 경우 IPA (아이소프로필 알코올)로 디텍터를 청소합니다.

디텍터를 청소하기 전 전원을 끄고 배터리를 분리합니다.

IPA 또는 다른 액체와 직접 접촉하지 않도록 방수 장갑을 착용합니다.

IPA를 디텍터에 직접 붓거나 분사하지 않습니다. IPA를 적신 직물 또는 부드러운 천을 사용하여 청소합니다.

IPA 또는 다른 액체가 디텍터에 들어가지 않도록 합니다.

청소 후 IPA가 완전히 건조될 때까지 기다립니다.

 6.2 보증 불가

H&abyz는 다음과 같은 환경에서 결함이 발생했을 경우 어떠한 책임도 지지 않습니다.

2) H&abyz의 사용자 매뉴얼의 내용에 따르지 않고 설치 혹은 사용으로 발생한 결함 및 손상

3) H&abyz의 사용자 매뉴얼에 명시된 입력 전원과 다른 입력 전원 사용으로 발생한 결함 및 손상

4) 구매자 혹은 제3자에 의한 부적절한 취급, 보관, 사용으로 인하여 발생한 결함 및 손상

5) 구매자 혹은 제3자가 제품의 소프트웨어를 임의로 수정하여 발생한 결함 및 손상

6) H&abyz에서 구매하지 않은 기기 또는 제품에 의하여 발생한 결함 및 손상

7) H&abyz의 사용자 매뉴얼에 명시된 환경 외의 제품의 설치 또는 사용으로 발생한 결함 및 손상

8) H&abyz의 보증 서비스에 포함되지 않는 구성품

9) H&abyz에 의해 인증되지 않은 제3자의 유지보수

10) 자연재해와 같은 불가항력 조건

6.3 보증 책임

H&abyz는 어떤 경우에도 구매자가 지불한 총 구매 가격(H&abyz 또는 H&abyz의 공인 대리점을 통한 구매)을 초과하는 손해비용 또는 기타 금액에 대하여 책임지지 않습니다. 또한 어떠한 상황에서도 사용상의 손실, 시간 손실, 데이터 손실에 대한 비용 및 상업상 손실 또는 기타 부수적 손해의 가능성에 대해 구매자가 내용을 통보 받았음에도 불구하고, 제품 사용 또는 사용 불능으로 인해 발생된 손실 또는 파생적 손해에 관하여 어떠한 책임을 지지 않습니다.

고객은 H&abyz에 반송할 제품에 대해서 물류로 운송할 책임이 있으며, 운송비는 고객사에서 선불처리가 되어야 합니다. 또한 운송 중에 발생한 제품의 분실 또는 파손에 대해 H&abyz는 책임을 지지 않습니다. H&abyz는 보증이 만료된 후 제품에 대하여 수리서비스를 진행할 경우 구매자에게 합리적인 수수료를 청구할 수 있는 권리가 있습니다.

H&abyz는 제품에 문제가 발생시 정상적인 사용 여부 및 원인에 대한 최종 결정을 내립니다. 공인대리인이나 구매자가 H&abyz의 조사 결과를 받아들이지 않으면 입증 책임은 그들에게 있습니다.

PART 2. 서비스 매뉴얼

이 서비스 설명서는 디텍터를 설정하는 방법에 대한 추가 지침을 제공합니다.

1. CS 관리자 지침

 1.1 디텍터 MAC & IP 주소 설정

 1) 디텍터 IP 주소 설정

디텍터를 연결하고 전원을 켭니다.

CS Manager를 실행하고 “IP_Change” 탭을 선택합니다.

변경하고자 하는 IP 주소를 2.2.22.10 ~ 2.2.22.101 중 선택하여 입력하고 “Set” 버튼을 클릭합니다.

(디텍터의 디폴트 IP 주소는 2.2.22.101 입니다)

팝업 창에 “IP Setting Done” 메시지가 표시될 경우 IP 변경에 성공, “작업 시간을 초과하였습니다” 메시지가 표시될 경우 IP 주소 변경에 실패한 것입니다.

IP 주소 변경 후 디텍터를 재부팅 해야 디텍터를 사용할 수 있습니다.

 2) 디텍터 MAC 주소 설정

디텍터를 연결하고 전원을 켭니다.

CS Manager를 실행하고 “MAC_Change” 탭을 선택합니다.

변경하고자 하는 MAC 주소를 입력하고 “Set” 버튼을 클릭합니다.

(디텍터의 디폴트 MAC 주소는 “1E:ED:19:27:1A:B3” 입니다.)

팝업 창에 “MAC Setting Done” 메시지가 표시될 경우 MAC 주소 변경에 성공, “작업 시간을 초과하였습니다.” 메시지가 표시될 경우 MAC 주소 변경에 실패한 것입니다.

MAC 주소 변경 후 디텍터를 재부팅 해야 디텍터를 사용할 수 있습니다.

 1. 2 Firmware, FPGA 업데이트

1) Firmware 업데이트

디텍터를 연결하고 전원을 켭니다.

CS Manager를 실행하고 “FW_Update” 탭을 선택합니다.

Update 버튼 왼쪽의 “…” 버튼을 클릭하여 업데이트 하고자 하는 Firmware 파일을 선택합니다.

“Update” 버튼을 클릭 한 후 팝업 창에 “Please Reboot” 메시지가 표시될 경우 FW 업데이트에 성공, “작업 시간을 초과하였습니다.” 메시지가 표시될 경우 FW 업데이트에 실패한 것입니다.

FW 업데이트 후 디텍터를 재부팅 해야 디텍터를 사용할 수 있습니다.

2) FPGA 업데이트

디텍터를 연결하고 전원을 켭니다.

CS Manager를 실행하고 “FPGA_Update” 탭을 선택합니다.

Update 버튼 왼쪽의 “…” 버튼을 클릭하여 업데이트 하고자 하는 FPGA 파일을 선택합니다.

“Update” 버튼을 클릭 한 후 팝업 창에 “Please Reboot” 메시지가 표시될 경우 FPGA 업데이트에 성공, “작업 시간을 초과하였습니다.” 메시지가 표시될 경우 FPGA 업데이트에 실패한 것입니다.

FPGA 업데이트 후 디텍터를 재부팅 해야 디텍터를 사용할 수 있습니다.

1. 3 RTC 설정

1) RTC 설정

디텍터를 연결하고 전원을 켭니다.

CS Manager를 실행하고 “RTC Set” 탭을 선택합니다.

Update 버튼을 누르면 Detector의 시간을 Window의 현재 시간으로 설정합니다.

“Update” 버튼을 클릭 한 후 팝업 창에 “RTC Set done” 메시지가 표시될 경우 RTC 설정에 성공, “작업 시간을 초과하였습니다.” 메시지가 표시될 경우 RTC 설정에 실패한 것입니다.

디텍터에 설정된 시간은 프로그램에 표시됩니다.

1. 4 Default IP 설정

1) Default IP 설정

디텍터를 연결하고 전원을 켭니다.

CS Manager를 실행하고 “Default IP” 탭을 선택합니다.

연결할 Detector의 IP 주소를 입력하고 “Set” 버튼을 누릅니다.

CS_Manager에서 디텍터로 연결을 시도할 때 Default IP에서 설정한 IP주소로 연결을 시도합니다.

2. 문제 해결

 2.1 LAN 연결 문제

 1) 전원 확인

데이터 케이블과 전원 코드가 제대로 연결되어 있는지 확인합니다.

전원 공급 장치의 전원이 켜져 있는지 확인합니다.

 2) PC 설정 확인

"인터넷 프로토콜 버전4(TCP/IP)"에서 IP 주소가 "2.2.22.102 ~ 2.2.22.254"로 설정되어 있는지 확인합니다.

 2.2 Auto Trigger(오토 트리거) 모드

Auto Trigger 모드를 사용하려면 몇 가지 요구 사항을 준수해야 합니다. 요구 사항이 충족되지 않으면 영상 획득에 실패할 수 있습니다.

영상 획득에 실패할 시, 다음과 같은 Auto Trigger(오토 트리거) 모드 요구 사항을 확인합니다.

최소 X-ray 조사 시간이 30msec로 유지되어 있는 지 확인합니다.

“Preview” 창에 “Ready for Exposure” 메시지가 표시될 때 X-ray 를 조사합니다. 메세지가 “Busy”일 경우 X-ray를 조사하지 마십시오.
