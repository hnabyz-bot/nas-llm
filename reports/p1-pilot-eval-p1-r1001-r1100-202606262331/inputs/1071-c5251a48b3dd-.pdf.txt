모양 및 구조 - 작용원리
엑스레이를 가시광선으로 변환하기 위한 Scintillator와 이를 데이터로 변환하기 위한 a-Si
TFT (또는 IGZO) 패널로 구성됩니다. TFT 어레이는 비정질 또는 무질서 상태에 있는 얇은 실리
콘층으로 구성되며, 실리콘은 그래프 용지의 그리드와 같이 고차 배열된 수백만 개의 트랜지스터
로 각인되어 있습니다. 이들 박막 트랜지스터 (TFT) 각각은 개별 픽셀을 구성하는 광 흡수 포토
다이오드에 부착되어 있으며, 광 다이오드로 들어온 가시광선은 강도에 따라 디지털 신호로 변환
되어 판독 가능한 디지털 이미지로 화면에 출력됩니다.

다. 소프트웨어 무선 연결
1) 디텍터 연결 방식
가) Wired: 유선 연결 방식
나) Wireless: 무선 연결 방식
디텍터는 지정된 SSID를 가지는 라우터를 검색하고 연결합니다.
다) AP: 무선 연결 방식
디텍터는 지정된 SSID, Channel, 보안 방식 및 Password 설정을 사용하여 AP를 제공하
며, 무선 통신을 제공하는 PC는 이 AP에 연결을 시도할 수 있습니다.
2) Connection Mode
소프트웨어의 상단에 위치한 Connection Config 버튼을 눌러 다음 화
면에 진입합니다.
Set 또는 Get 버튼을 눌러 디텍터의 Connection Mode를 설정하거나 조회할 수 있습니다.
디텍터는 5가지의 Connection Mode를 제공하며 각 모드는 아래와 같은 방식으로 동작합니다.
가) Ethernet Only: Wired(유선) 모드입니다. Gender Cable을 통해서만 통신할 수 있습니다.
나) Allow Wireless: Wireless 허용 모드입니다. 기본적으로 Gender Cable을 통해서 통신할
수 있으며, Cable이 물리적으로 분리되는 경우 자동으로 Wireless 연결을 시도합니다.
(단, Wireless 연결 정보가 유효해야 합니다.)
다) Allow AP: AP 허용 모드입니다. Gender Cable을 통해서 통신할 수 있으며, Cable이
물리적으로 분리되는 경우 자동으로 Host AP를 시도합니다.
(단, AP 설정값이 유효해야 합니다.)
라) Wireless Only: Wireless 모드입니다. 디텍터의 전원이 On된 시점부터 상시로 Wireless
상태를 유지합니다. 설정된 Wireless 연결 정보를 사용하여 연결에 성공할 때까지 라우터에
연결을 지속적으로 시도합니다 . Gender Cable이 부착되더라도 Wired 모드로 전환되지 않
습니다.
(만약 설정된 Wireless 연결 정보가 유효하지 않다면, 디텍터는 통신이 불가능한 상태가 됩니
다. 이러한 경우 디텍터의 AP 버튼을 5초 이상 길게 눌러 Connection Mode를 Allow AP로
초기화한 뒤 Gender Cable 또는 AP를 사용하여 설정값을 확인하십시오.)
마) AP Only: AP 모드입니다. 디텍터의 전원이 On된 시점부터 상시로 AP 상태를 유지합니다.
설정된 AP 정보를 사용하여 Host AP를 수행합니다. 사용자는 디텍터의 SSID를 무선 네트워
크 목록에서 찾아 연결을 시도할 수 있습니다. Gender Cable이 부착되더라도 Wired 모드로
전환되지 않습니다.
(만약 설정된 AP 정보가 유효하지 않거나 Host AP에 실패한 경우 디텍터는 통신 불가능한
상태가 될 수 있습니다. 이러한 경우 디텍터의 AP 버튼을 5초 이상 길게 눌러 Connection
Mode를 Allow AP로 초기화한 뒤 Thether Cable을 사용하여 설정값을 확인하십시오.)

3) IP
디텍터는 모든 Connection Mode에 대해서 하나의 IP를 사용하며 이 페이지에서 IP를 변경할
수 있습니다. IP의 초기값은 192.168.0.54입니다.
디텍터는 255.255.255.0의 서브넷 마스크를 사용하기 때문에 192.168.0은 고정값으로 두고 마지
막 옥텟만을 변경하는 것을 권장합니다.
(단, 마지막 자리에 0, 1 및 255는 사용할 수 없습니다.)
디텍터의 IP가 정상적으로 변경되었다면 현재 연결은 즉시 끊어지며 디텍터는 새로운 IP로의 통
신을 개시하게 됩니다.
