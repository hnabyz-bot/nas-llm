1. Summary
- This clinical trial report is intended to ensure the clinical safety of GoStitching App,
GoStitching Web. For this, clinical data were collected, evaluated and analyzed.
1.1 Determination of the benefit/risk profile in the intended target group
- Intended target group for user: GoStitching App, GoStitching Web shall be used by
physician. There is no risk due to the use of this software in the user’s viewpoint, but
there is an benefit that GoStitching App, GoStitching Web can provide stitched medical
image easily to promote intended purpose.
1.2 Demonstration of acceptability of that profile based on the state of the art in the medical
fields concerned.
- State of the art in the medical fields: The technical aspect of the "Automatic Image
Stitching Software" that has been used around the world for over 10 years is only to use
the well known image stitching algorithm. Technologies other than image stitching
algorithm do not currently exist, and all international research institutions use the same
technical principles.
- And the technologies used for this software has been already known widely in medical
image areas. There are sufficient literature to verify the safety and effectiveness of the
software, and software with the same principles as GoStitching App, GoStitching Web are
sold, and the equivalence of these softwares is compared and the safety and validity are
proved through the literature, it is not necessary to conduct clinical investigation. The
relevant technology is a typical one that has been used since decades years ago.

- Operating environment (minimum recommendation

2) CPU: Intel i5 or above
3) RAM: 4GB or above
4) HDD: 500GB or above
5) Monitor: A monitor capable of expressing a resolution of 1920 X 1080 or higher
6) Network : 100Mbps or above for DICOM Network
- Software main functions
1) DICOM format video support: DICOM 3.0 standard is supported, and DICOM video
load, save, send, and receive functions are provided.
2) Non-DICOM format video support: Provides Load and Save functions for non-DICOM
(Tiff, Raw) images as well as DICOM images.
3) Image conversion: It provides a function to transform the image. Image transformation
functions include Windowing Width/Level adjustment, Rotate, Flip, and Cropping
functions.
4) Image matching: Provides a function of matching multiple input images into one image
based on the overlapping area. The matching function includes a manual matching
technique in which the user directly finds and inputs the overlapping area, and an
automatic matching function that automatically determines the overlapping area through an
algorithm.

Medical Image Stitching Using Hybrid Of Sift & Surf Techniques
Abstract — Image stitching is a technique that combines two or more images from the same scene
to obtain a panoramic image. Image stitching is used in Medical system for stitching of X-ray
images. As the flat panel of X-ray system cannot cover all the parts of a body. So stitching of
Medical images can be done. Stitching of images basically includes two main parts – Image
Matching and Image Blending. For Image Matching the two algorithms SIFT and SURF are used.
This paper presents a technique using hybrid of SIFT and SURF. As SIFT is slow process and not
suitable at illumination changes while it is invariant to scale changes, rotation and affine
transformations whereas SURF is having illumination property and having high computational speed.
So by combining both methods, a new method can be generated that creates panoramic image having
better features.
