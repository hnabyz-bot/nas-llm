A summary describing the plan for providing validated software updates and patches as needed throughout the lifecycle of the medical device to continue to assure its safety and effectiveness. The FDA typically will not need to review or approve medical device software changes made solely to strengthen cybersecurity.

We make release and design changes in accordance with software development procedures(Doc. No. AQP702).

The operator can update firmware automatically on network connection / wireless network setting. 

(See below or refer to User Manual)

Detector Connection

In ‘’Settings’’ tab, enter IP address of the connected detector into “Connection IP” and click “Read” for detector information.

Update

Click “…” next to “Update file” to attach the file to update and click “Update”. Update completion is shown in “Log” tab.

Network Connection

Configuration tab shows current network connection information.

You can change detector IP, MAC address, netmask, country and wireless network settings. Click “Update” after setting and changing.

Wireless Network Settings

Click “Scan” next to AP list to see the wireless network list that can be connected.

Select the network that you want to connect, enter the password, and click “Update”.

Click “Read” to check the wireless network information.

Click “Clear AP Info” and “Update” in order to initialize the wireless network settings.

A summary describing controls that are in place to assure that the medical device software will maintain its integrity (e.g. remain free of malware) from the point of origin to the point at which that device leaves the control of the manufacturer; and

This device restrict random access by setting id / pw when connecting to Wireless network.

And transmission data should be protected by applying encryption such as WPA, WPAPSK, WPA2, WPA2PSK, WPAPSK+TKIP, etc.

Also, this device provide the calibration for the quality of the detector. It developed to perform quality assurance operations on the detector. This calibration function shall be performance once every one year for stabilized detector condition.

Please refer to the following steps for the calibration.

Dark Image Acquisition

Click "Get Dark" to get a Dark image on the "Calibration" tab. The "Preview" window appears and acquires the Dark image. The acquired Dark image will be displayed in the image file list.

H&Abyz recommends to use the "Average Dark" option to enhance image performance.

If you select the "Average Dark" option, Dark images will be automatically acquired to the number of "total dark frame" values.

Dark image in AED Mode (Auto Exposure Detection Mode) is saved in the below location.

C:\XAS_W\Cal_0\CalType0-AED\

Dark image in Normal Mode is saved in the below location.

C:\ XAS_W \Cal_0\CalType0-NorTrg\

Bright Image Acquisition and Creation of calibration Point Image

According to the Trigger Mode (Trigger Mode), the bright image is acquired in different ways.

In AED Mode (Auto Exposure Detection Mode), click the “Get Bright” button of the software to acquire the bright image. The bright images must be acquired at twenty calibration points. The acquired bright images are shown in the image file list.

In Normal Mode, without clicking the “Get Bright” on the software and uses the Switch Box Exposure switch to acquire the bright image. The bright images must be acquired at twenty calibration points. The acquired bright images are shown in the image file list.

H&Abyz recommends twenty calibration points such as written below. Some artifact can be seen on images if the calibration points are not sufficient.

In AED mode (Auto Exposure Detection Mode), each bright image must be acquired at intervals of 30 seconds.

In AED mode (Auto Exposure Detection Mode), X-ray shall be exposed within 30 seconds once the “Acquisition” button is clicked. Otherwise, the action will be cancelled.

In Normal Mode, the image is acquired using Switch Box Exposure switch, not clicking “Get Bright” button.

Do not expose X-ray when “Busy” message is displayed in “Preview” pop-up window. Any Images can’t be acquired during “Busy” state although X-ray is exposed. Expose X-ray when “Ready for Exposure” is shown up in “Preview” window.

When acquiring the bright image, the name of the image file must be Bright01.raw ~ Bright20.raw. When you create the bright image file, the Cal point image "*_OC_xxxx.raw" is automatically created. The acquired bright images are shown in the image file list.

BPM Generation

Click “Generate BPM” in BPM tap, and click “OK” in pop-up to finish generating the BPM.
