Introduction

Purpose of the Document

The purpose of this document is to outline the Usability Validation Plan for the Digital Flat Panel X-ray Detector GT1717C, GT1717CP. This plan ensures that the device meets usability requirements, provides a safe and effective user experience, and complies with applicable standards and regulations.

Scope

This plan applies to all usability validation activities for the Digital Flat Panel X-ray Detector GT1717C, GT1717CP, focusing on safety-critical tasks, user interface evaluation, and error minimization. It includes activities to evaluate the interaction of users with the device under realistic conditions.

Product overview

Product Description:

The GT1717C/GT1717CP are wired digital flat panel detectors that have been designed for faster, more streamlined approach to digital radiography systems. The detector consists of TFT and Scintillator (CsI) and an electronic device that converts data read through it into digital signals. The detectors support an auto exposure detection technology that allows the detectors to be used without generator integration.

The flat panel sensors of GT1717C/GT1717CP are fabricated using thin film technology based on amorphous silicon technology. Electronically, the sensors are much like conventional photodiode arrays. Each pixel in the array consists of a light‐sensing photodiode and a switching Thin Film Transistor (TFT) in the same electronic circuit. The amorphous silicon photodiode is sensitive to visual light, and its response curve has almost the same performance as human vision. The sensitivity of amorphous silicon photodiodes peaks in green wavelengths, well matched to scintillators such as CsI. It also has excellent linearity according to the exposure dose.

Key Features

Read the user manual carefully before using the product.

Connect the AC power cable to the POE adapter to supply AC power.

DC voltage from the POE adapter is supplied to the detector via the LAN cable.

The detector has a power button that can be used to turn the power on and off.

A handle is provided as an optional accessory to make it easier to move the product.

Detector calibration can be performed using software installed on a PC.

An indicator (LED) allows you to check the Ethernet connection status.

X-rays can be taken to identify the body's anatomical structure.

Familiarity with product maintenance methods is required.

Overview of Usability Engineering Process

Usability Goals

Safety: Ensure safe operation by minimizing user errors.

Effectiveness: Enable users to perform imaging tasks efficiently.

User Satisfaction: Provide an intuitive and user-friendly interface.

Design Inputs and Validation Linkage

The usability validation activities directly address design inputs, including:

User interface layout and clarity: Ensure all buttons and indicators are clearly labeled and positioned for easy access.

Safety-critical features: Verify the effectiveness of safety controls, such as dose adjustment and operational state LEDs.

Task workflow: Evaluate typical user workflows to minimize complexity and reduce errors.

Summary of Formative Evaluation Results

Usability Formative Evaluation was conducted through Usability Formative Evaluation Plan(Doc No. A-UFEP-CYN) and Usability Formative Evaluation Report(Doc No. A-UFER-CYN), through which the following key findings were discovered and improvements were implemented.

Methods Used: Iterative prototyping and user testing with radiology technicians and clinicians.

Key Findings:

The user manual must be written in an easy-to-understand manner.

Each button and status LED should be easy to identify.

When connecting components to the detector, there should be no confusion or misconnection.

The product should be easy to transport.

The calibration procedure should be clearly and thoroughly explained.

The detector should have markings identifying the patient's imaging area.

The product maintenance procedure should be simple and concise.

Improvements Implemented:

The user manual is written in an easy-to-understand and concise manner.

The Ethernet connection status can be confirmed via an indicator (LED).

The power button's LED indicates the power ON/OFF status.

The product is designed to prevent incorrect connection of components, and the user manual contains detailed connection instructions.

The product's calibration procedure is clearly described in the user manual.

The detector has markings that identify the patient's imaging area. How to maintain the product is explained in an easy-to-understand manner in the user manual.

Overview of Summative Evaluation Plan

The summative evaluation will test the final design in real-world conditions with representative users to validate usability and safety goals. It will focus on critical tasks, including setup, dose adjustment, and image capture.

Validation Strategy

Validation Objectives

Verify that the product can be used safely and effectively by intended users in the intended use environments.

Identify and mitigate any residual usability risks.

Scope of Validation

User Groups: Radiologists, technicians, and clinicians with varying levels of expertise.

Use Environments: Hospital settings, mobile clinics, emergency response units.

Key Tasks:

Check the user manual to ensure familiarity with product operation.

Check how to connect components to the detector.

Check indicator (LED) displays and status.

Check product transport.

Check understanding of the calibration procedure.

Check the detection area identification markings on the detector.

Check familiarity with product maintenance instructions.

Validation Approach

Conduct usability tests under simulated clinical environments to replicate real-world conditions.

Use quantitative metrics (e.g., task completion time, error rates) and qualitative feedback (e.g., user satisfaction surveys) for evaluation.
