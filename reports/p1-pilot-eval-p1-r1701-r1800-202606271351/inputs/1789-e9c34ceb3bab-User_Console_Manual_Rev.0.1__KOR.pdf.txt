Digital Radiography Console Software feel-DRCS

1. 소개
feel-DRCS 는 H&abyz 사의 Digital Radiography Control Software 의 제품명
입니다.
feel-DRCS 는 flat-panel 또는 CCD 등의 DR detector 로부터 영상 획득 및
임상적용을 위한 빠른 속도의 영상처리, 그리고 PACS 연동을 위한 DICOM
서비스를 제공하는 DR 콘솔 또는 DR 컨트롤 소프트웨어입니다.
이 매뉴얼은 feel-DRCS 소프트웨어를 설치 및 동작 하기 위한 내용을 정리
되어 있습니다.
주식회사 에이치앤아비즈(H&abyz)
경기도 용인시 처인구 남사면 처인성로 41-16 2 동 1 층 일부
(우) 17118

Digital Radiography Console Software feel-DRCS

2. 목적
feel-DRCS 는 FS-MLW(Faster Specialized Multi Layered Wavelet) 기술을 기반
하여, 각종 detector 나 X-Ray Generator 와 integrated 된 Operation
Console System 소프트웨어입니다.
feel-DRCS 는 절대적인 사용자 입장에서 개발된 제품으로, 다년간 임상사이
트인 병원에서 디자인 되었으며, 서비스를 위한 DR 제조사들의 서비스 엔
지니어들에 의해 완성된 제품입니다. 그러므로 쉽고, 간결하면서도 다양한
기능들이 포함된 최상의 User interface 로 개발된 DR 콘솔 소프트웨어입니
다.
feel-DRCS 는 feel-TK 를 기반으로 내장되어 있기 때문에 보다 빠른 속도로
노이즈와 각종 artifact 제거되고, 보다 선명한 생체조직을 볼 수 있습니다.
그리고 독자적인 calibration 기능이 내장되어, 모두 종류의 detector 에 대
한 기본 calibration data(offset image and gain image)을 설정할 수 있습니
다. 그리고 defect view 를 제공하여 진행성 defect 들을 처리할 수 있습니다.
feel-DRCS 는 전세계적으로 판매되고 있는 각종 디텍터들에 대해서 이미 소
프트웨어 검증 및 판매가 되어서 임상필드에서 사용되고 있습니다. DR 제조
사가 사용하고자 하는 디텍터를 손쉽게 설정에서 타입만 변경하면 바로 콘
솔시스템의 소프트웨어로 사용이 가능합니다. 그리고 통합 DR 시스템인 X-
ray 의 generator operation panel 을 소프트웨어 내부에 integration 되어 있
어, APR, AEC, Ready/Exposure 등을 제어가 손쉽게 가능합니다.
feel-DRCS 는 각종 X-ray generator 와 연동이 되어, APR 기능 등을 지원하여,
하나의 콘솔 소프트웨어에서 기존의 Operation Panel 의 역할을 같이 수행
할 수 있게 하여 분리된 DR 제품이 아닌 하나의 통합된 DR 제품을 지원합
니다.
feel-DRCS 는 단순히 소프트웨어를 공급하는 것이 아니라, 다년간 경험했던
노하우를 DR 제조사에게 전달해 드리며, 동시에 컨설팅까지 같이 지원해
줌으로써 조력자의 역할을 다 할 것입니다.

2-1. 사용용도
Feel-DRCS 는 방사선사 및 의사들이 일반 x-ray 영상을 획득하기 위한 용
도로 사용된다. 그래서 병원에서 근무하는 의사 및 방사선사 외에는 사용
할 수 없다. 또한 이 소프트웨어는 서비스 엔지니어에 의해서만 설치되고
방사선기사 및 의사에 의해서만 사용되어진다.
올바른 사용 및 안전을 위해 사용 설명서의 내용을 완전히 숙지한
후에 적법한 자격을 갖춘 사람 (의사, 의료 방사선 전문 기술자,
의료용 X-ray 기술자)만 이 제품을 사용할 수 있도록 하십시오.
이 의료 기기는 단독으로 사용할 수 없습니다. 따라서 이 제품은
X-ray 디텍터와 연결해야 합니다. 이 소프트웨어는 X-ray 발생장치
시스템의 노출 조건에서 작동합니다.
이 제품은 의료 기기로 해당 병원의 X-ray 방사선 촬영을 위한 디지
털 X-ray 촬영 콘솔 소프트웨어로만 사용해야합니다.
2-2. 심볼 라벨

부상을 초래할 수 있는 내용을 나타냅니다.

3. 요구 사항
3-1. Workstation
 Processor: Intel Core i3 이상(i5 이상 추천)
 RAM : 4GB RAM 이상(8GB 이상 추천)
 HDD: At least 40GB hard disk(가능하다면 7,200 RPM 이상 속도의 제품).
데이터백업 및 안정성을 위하여 1 개의 HDD 를 논리적으로 파티션을 나눈
것이 아니라, 2 개의 물리적으로 분리된 HDD 를 장착하여 1st HDD: System
and Software, 2 HDD: Image Data Storage 용 사용할 것을 추천.
 Network Card : 10/100/1000Mbps Ethernet Network Card
 Video graphic adapter: 해상도는 최소한 1280 X 900 pixels 이며 칼라 해
상도는 True Color mode(no shard memory), DVI interface connector
 OS : Windows 7/8.1/10
3-2. Monitor
 1600 X 1200 or 1920 X 1280 추천(최저 해상도 1280 X 900 이상)
4 : 3, 16 : 9, 16 : 10 비율 모두 지원
 High fidelity of grayscale and optimized luminance distribution
만약 고휘도, 고해상도의 모니터를 사용하고자 한다면, 의료용 LCD
Monitor(B/W or Color)를 선택하기로 추천함. 그리고 B/W 모니터를 선택하
여도 현재 소프트웨어가 회색조 베이스에서 만들어져 있으므로 사용하는데
전혀 문제가 되지 않음.
영상 QC 를 위해서는 의료용 Monitor 에서 하기를 추천한다. 그 이유는 DR
이기 때문에 촬영 후 의료용 monitor 에서 판독을 하는 이유로 같은 밝기
에서의 영상 조절이 이루어져야 하기 때문이다. 그리고 대부분의 의료용
monitor 들은 각 국가별 의학기준에 근거하여 제작되어 있으므로 사용에
대한 제약이 거의 없다.

6. feel-DRCS 사용법
6-1. 로그인
프로그램을 시작합니다.
바탕화면에서 feel-DRCS 단축아이콘을 더블 클릭하여 프로그램을 실행한다.
로그인 화면에 사용자 이름(혹은 아이디)과 비밀번호를 입력하고 OK 버튼
을 선택하여 프로그램을 시작한다.
로그인을 하고 나면 최초에 각종 연결된 디텍터 및 generator 와의 연결을
확인하여 화면의 각 부위의 연결된 색이 표시된다.
- Company 또는 회사이름, SUPERVISIOR 등의 사용자 별에 따라
기능 권한이 차이가 나게 된다.
- 일반사용자 및 일반관리자는 병원에서 사용가능 한 계정이며,
SUPERVISIOR 계정은 A/S 엔지니어 사용자 계정임.
- 로그인 후 장비의 연동에 따라 잠시 화면이 멈춰 있게 보일 수도
있다

Digital Radiography Console Software feel-DRCS

6-2. Worklist
설정된 Worklist 서버에서 촬영목록을 읽어와 화면에 표시한다. 수동으로 입력할 수도
있다.
리스트에서 환자정보를 더블 클릭하거나 Exam 버튼을 클릭하여 촬영모드로 이동할
수 있다.
6-2-1. Main menu
① EXAM : 아래 리스트에서 선택된 검사에 대해서 촬영모드로 전환
② DELETE : 선택된 촬영 검사 목록을 삭제한다.
③ REFRESH : DICOM Worklist 가 지원되는 경우에 목록 갱신한다.
④ ER : 응급촬영을 실시한다.
⑤ CODE : RIS 코드를 추가 및 수정하기 위하여 code manager 를
불러온다
ER 의 경우에 사전에 코드 매니저에서 ER001 special code 를 사전
에 설정해 두어야 아이디, 이름 등 자동으로 생성되면서 촬영 모드
로 넘어가게 된다. 이때 검사코드에 촬영부위, 촬영조건 등을 기입
해 놓은 것이 촬영설정이 된다.

Digital Radiography Console Software feel-DRCS
21
6-2-2. Search Option
리스트에 표시되는 항목을 설정할 수 있다.
환자정보(ID, Name, Access No.)를 이용하여 특정환자만 표시하거나
Schedule, Modality, State, D.O.B(Date Of Birth) 항목을 이용하여 특정
조건에 맞는 환자만 표시할 수 있다.
6-2-3. Refresh
워크리스트 서버에 연결이 가능할 경우 서버에서 환자정보를 가져올 주기를
설정할 수 있다.
AUTO 가 보이게 되는 경우에는 우측에 설정된 시간 주기로 자동으로
리스트를 갱신하게 된다.
만약 자동 리스트 갱신을 원하지 않는 경우에는 AUTO 버튼을 눌러서
MANUAL 로 보이게 하면 원하는 경우에만 refresh 하게 된다. 이때 refresh
버튼은 6-2-1. 에 있는 REFRESH 버튼을 클릭하여 수동 목록 갱신을 할 수
있다.
6-2-4. New SET
각종 설정(Body part, View position, X-ray exposure
condition, Image processing parameter, Annotation etc)을 수정하거나
추가하기 위하여 NewSET 을 설정한다. 사용법은 Generator 부분을
참조한다.

6-2-5. Exam List
Option 에 설정된 조건에 따라 해당되는 목록이 표시된다.
